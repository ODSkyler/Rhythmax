<h1 align="center">🎧 Rhythmax</h1>

<p align="center">
A lightweight open-source music player designed for both <b>offline playback</b> and <b>online streaming</b>.
</p>

<p align="center">
Built with Flutter.
</p>

<hr>

<h2>✨ Features</h2>

<ul>
<li>🎵 Offline music playback</li>
<li>🌐 Online music streaming</li>
<li>📝 Lyrics support</li>
<li>📀 Album and artist metadata</li>
<li>📱 Clean and responsive UI</li>
<li>🔀 Queue, shuffle, and repeat controls</li>
</ul>

<h2>🚀 Getting Started</h2>

<h3>Requirements</h3>

<ul>
<li>Flutter SDK</li>
<li>Android Studio or VS Code</li>
<li>Git</li>
</ul>

<h3>Clone the repository</h3>

<pre>
git clone https://github.com/ODSkyler/Rhythmax.git
cd Rhythmax
</pre>

<h3>Run the project</h3>

<pre>
flutter pub get
flutter run
</pre>
